seafile安装与备份恢复

一、准备工作

1.安装docker
2.安装docker-compose
3.安装ossutil【非必须】

二、安装

docker-compose.yml

version: '2.0'
services:
  db:
    image: mariadb:10.5
    container_name: seafile-mysql
    environment:
      - MYSQL_ROOT_PASSWORD=mysql密码  # Requested, set the root's password of MySQL service.
      - MYSQL_LOG_CONSOLE=true
    volumes:
      - /home/ubuntu/backup-mysql:/var/lib/mysql  # Requested, specifies the path to MySQL data persistent store.
    networks:
      - seafile-net

  memcached:
    image: memcached:1.5.6
    container_name: seafile-memcached
    entrypoint: memcached -m 256
    networks:
      - seafile-net

  seafile:
    image: seafileltd/seafile-mc:latest
    container_name: seafile
    ports:
      - "8000:80"
#      - "8001:443"  # If https is enabled, cancel the comment.
    volumes:
      - /home/ubuntu/backup:/shared   # Requested, specifies the path to Seafile data persistent store.
    environment:
      - DB_HOST=db
      - DB_ROOT_PASSWD=mysql密码  # Requested, the value shuold be root's password of MySQL service.
      - TIME_ZONE=Asia/Shanghai # Optional, default is UTC. Should be uncomment and set to your local time zone.
      - SEAFILE_ADMIN_EMAIL=邮箱 # Specifies Seafile admin user, default is 'me@example.com'.
      - SEAFILE_ADMIN_PASSWORD=用户名     # Specifies Seafile admin password, default is 'asecret'.
      - SEAFILE_SERVER_LETSENCRYPT=false   # Whether use letsencrypt to generate cert.
      - SEAFILE_SERVER_HOSTNAME=网址 # Specifies your host name.
    depends_on:
      - db
      - memcached
    networks:
      - seafile-net

networks:
  seafile-net:

需要修改的地方:

MYSQL_ROOT_PASSWORD=mysql密码  # 如 cpzt123
DB_ROOT_PASSWD=mysql密码  # 如 cpzt123
SEAFILE_ADMIN_EMAIL=邮箱  # 如 cpzt@gmail.com
SEAFILE_ADMIN_PASSWORD=用户名  # 如 cpzt123
SEAFILE_SERVER_HOSTNAME=网址  # 如 127.0.0.1:8000 ,这里的端口号一定要加上

三、备份与恢复

1.数据备份

backup.sh

#!/bin/bash

BASE_DIR="$HOME/test"

DIRNAME="seafile_backup_`date +%Y%m%d%H%M`"
ROOT_DIR="$BASE_DIR/$DIRNAME"
DB_DIR="$ROOT_DIR/db"

SRC_DIR="$HOME/backup"

helpFunc() {
  echo "Usage:"
  echo "backup.sh [-d] [-u]"
  echo "Description:"
  echo "-d, 只保留打包文件"
  echo "-u, 上传oss"
}

IS_DELETE_DIR=false;
IS_UPLOAD=false;

while getopts 'h d u' OPT; do
  case $OPT in
    d) IS_DELETE_DIR=true;;
    u) IS_UPLOAD=true;;
    h) helpFunc & exit 0;;
    ?) helpFunc & exit 0;;
  esac
done

echo $IS_DELETE_DIR;

# 创建目录
echo "创建备份目录: $ROOT_DIR"
mkdir $ROOT_DIR
mkdir $DB_DIR

echo "备份数据库"
cd $DB_DIR
docker exec -it seafile-mysql mysqldump -uroot -pchim --opt ccnet_db > ccnet_db.sql
docker exec -it seafile-mysql mysqldump -uroot -pchim --opt seafile_db > seafile_db.sql
docker exec -it seafile-mysql mysqldump -uroot -pchim --opt seahub_db > seahub_db.sql

echo "备份seafile数据"
sudo cp -rf $SRC_DIR/seafile $ROOT_DIR


echo "压缩打包"
cd $BASE_DIR
sudo tar zcf "$DIRNAME.tar.gz" $DIRNAME

if [ "$IS_DELETE_DIR" = true ]
  then sudo rm -rf $ROOT_DIR & echo "删除原始文件"
fi


if [ "$IS_UPLOAD" = true ]
  then ossutil64 cp $DIRNAME.tar.gz oss://你的ossbuckt/ & echo "上传oss成功"
fi

echo "完成备份"

exit 0

需要修改的地方:

if [ "$IS_UPLOAD" = true ]
  then ossutil64 cp $DIRNAME.tar.gz oss://你的ossbuckt/ & echo "上传oss成功"
fi

# 替换 你的oosbuckt

使用：

./backup.sh -d -u

2.恢复

recover.sh

#!/bin/bash

ROOT_DIR=$(cd $1; pwd)


if [ -d $ROOT_DIR ]
   then echo "$ROOT_DIR"
else
   echo "目录不存在"
   exit 1
fi

DB_DIR="$ROOT_DIR/db"


echo '恢复数据库'
cd $DB_DIR
docker cp ccnet_db.sql seafile-mysql:/tmp/ccnet_db.sql
docker cp seafile_db.sql seafile-mysql:/tmp/seafile_db.sql
docker cp seahub_db.sql seafile-mysql:/tmp/seahub_db.sql

docker exec -it seafile-mysql /bin/sh -c "mysql -uroot -pchim ccnet_db < /tmp/ccnet_db.sql"
docker exec -it seafile-mysql /bin/sh -c "mysql -uroot -pchim seafile_db < /tmp/seafile_db.sql"
docker exec -it seafile-mysql /bin/sh -c "mysql -uroot -pchim seahub_db < /tmp/seahub_db.sql"

echo '恢复seafile数据'

cd $ROOT_DIR
echo `pwd`
sudo cp -rf seafile/* $HOME/backup/seafile/

使用：

./recover.sh -d 备份文件目录
